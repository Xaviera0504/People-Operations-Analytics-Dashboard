# People-Operations-Analytics-Dashboard

## Overview
A sample Excel dashboard demonstrating an approach to tracking key internal workforce metrics for People Operations analysis.

## Purpose
This dashboard shows how to visualize and analyze:
- Departmental headcount distribution
- Employee retention and attrition rates
- Engagement and satisfaction trends by department
- Performance distribution across workforce
- Employment type composition (Full-Time, Contract, Part-Time)

## Contents
- `Cleaned_HR_Data_Analysis.xlsx` - Dashboard with 5 key metrics and 5 visualizations

## Key Metrics
1. Headcount by Department
2. Employee Status (Active vs. Terminated)
3. Engagement & Satisfaction by Department
4. Performance Distribution
5. Employee Type Mix

## How It Works
The dashboard translates raw employee data into clear metrics and charts that enable data-driven People Operations decisions: where to allocate resources, which departments need engagement support, and how workforce composition impacts strategy.
